# TCC-ETECAMP-MARTOPIA
O Projeto Martopia foi desenvolvido durante o 3º ano do ensino médio integrado ao técnico em Informática para Internet, com o propósito de modernizar e facilitar a dinâmica interna da escola. O trabalho surgiu a partir da necessidade de aprimorar a organização e a comunicação em atividades escolares, oferecendo uma solução prática e acessível para estudantes e professores. A proposta central do projeto é criar uma plataforma digital intuitiva, que permita gerenciar informações, eventos e projetos pedagógicos de forma eficiente. O desenvolvimento contou com a dedicação e colaboração dos alunos do 3º ano, que aplicaram conhecimentos técnicos adquiridos ao longo do curso para construir uma ferramenta útil, funcional e alinhada às demandas reais da comunidade escolar.
